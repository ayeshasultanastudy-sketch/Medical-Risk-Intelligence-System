🚀 Project Overview

The Medical Risk Intelligence System predicts multiple health conditions using trained ML models and rule-based logic. It provides an intuitive interface for analyzing patient data and generating risk insights instantly.

🔍 Key Capabilities:
🧬 Tumor (Cancer) Prediction
❤️ Overall Health Risk Scoring
📊 Risk Probability Estimation
⚡ Real-time Interactive Dashboard

⚙️ Tech Stack
🐍 Python
📊 Pandas & NumPy
🤖 Scikit-learn
🌐 Streamlit
💾 Joblib

🧪 Features
🧠 Multi-model AI prediction system
📊 Real-time analytics and visualization
📈 Probability-based risk estimation
🖥️ Clean and responsive UI (Streamlit)
🔀 Hybrid system (ML + Rule-based logic)
📊 Machine Learning Models
🌳 Random Forest Classifier
🧾 Rule-based Risk Engine
🧬 Breast Cancer Dataset (sklearn)
🧪 Synthetic Health Dataset
